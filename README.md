# Weather Dashboard

## About
A responsive Weather Dashboard built using HTML, CSS, and JavaScript. It provides real-time weather information for any city using the WeatherAPI.

## Features
- Search weather by city name
- Current temperature
- Weather condition
- Weather icons
- Wind speed
- Humidity
- Feels Like Temperature
- Rain Chance
- Interactive Map
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- WeatherAPI
